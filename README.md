# Employee Database Project (SQL Server)

###  Overview
This SQL project manages employee data across multiple departments.  
It demonstrates:
- Database creation  
- Table relationships (Foreign Keys)  
- Data insertion  
- Joins, Views, Aggregations  

###  Tables
- Departments  
- Employees  
- Salaries
